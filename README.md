# Student_Placement_Prediction
The Machine learning project focused on whether student will be placed or not on the basis of academic scores and other related features.
A complete Machine Learning Pipeline is been followed-
1. Data Collecting
2. Data PreProcessing & Cleaning
   - Handling missing values
   - Removing duplicates
3. Exploratory Data Analysis (EDA)
   - Visualization
   - Department wise placement analysis
   - Correlation heat map
4. Feature Engineering
5. Model Training
   - Logistic Regression
   - Decision Tree
   - Random Forest
   - Navie Bayes
   - Support Vector Machine
6. Model Evaluation
   - Accuracy_Score

## Results 
Above all Model Training and Evaluation the best performed model is "The Random Forest" with best accuracy among all the models.
